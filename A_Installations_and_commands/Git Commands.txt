/* Multiple github accounts using ssh keys */
    > mkdir .ssh (if not present)
    > cd .ssh
    > ssh-keygen
    > cat id-rsa.pub (copy the response)
    
    > after that go to github.com and generate a ssh key using above response

    > notepad config
        > #Account1
          Host github.com
            HostName github.com
            IdentityFile ~/.ssh/id_rsa
